# 🚀 JobApplyAI-Platform

**The industry’s first glassmorphic, AI-driven job application engine designed to automate your career journey.**

JobApplyAI is a high-performance, autonomous platform that bridges the gap between candidates and recruiters using advanced AI simulations, real-time ATS scoring, and a state-of-the-art design language.

![JobApplyAI Preview](https://github.com/BLACKDEVIL4114/JobApplyAI-Platform/raw/main/preview.png)

## ✨ Key Features

- **🎯 Dynamic ATS Scoring**: Instantly analyze your resume against 30+ industrial technical keywords with an intelligent density-based algorithm.
- **🤖 Autonomous Auto-Apply**: Experience the future of job hunting with a high-scale application simulator that includes randomized recruiter feedback loops ('Selected', 'Interviewing').
- **🖼️ Premium Glassmorphic UI**: A stunning, high-contrast interface built with **Framer Motion** for smooth, buttery transitions and **Tailwind CSS** for a tailored dark-mode experience.
- **🔐 Secure Auth Vault**: Production-ready authentication architecture using **Supabase**. Supports standard Email/Password and high-fidelity **Google Sign-In**.
- **📊 Command Center**: A centralized dashboard to track your application metrics, success rates, and real-time job matches.

## 🛠️ Tech Stack

- **Frontend**: React, TypeScript, Vite
- **Styling**: Tailwind CSS, Lucide Icons
- **Animations**: Framer Motion
- **Database/Auth**: Supabase (Architecture ready)
- **Backend Mocking**: Custom Vite Middleware API

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/BLACKDEVIL4114/JobApplyAI-Platform.git
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Run the development server**:
   ```bash
   npm run dev
   ```
4. **Connect your Database**:
   Rename `.env.example` to `.env` and add your Supabase credentials to enable real-world authentication.
